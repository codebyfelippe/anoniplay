## 🎬 Sobre o AnoniPlay

O **AnoniPlay** é uma plataforma de filmes e séries online criada para oferecer uma experiência simples, moderna e acessível, contendo apenas **um anúncio por conteúdo**, evitando excesso de publicidade e proporcionando uma navegação mais agradável aos usuários.

O projeto é totalmente **Open Source**, permitindo que qualquer pessoa contribua com ideias, melhorias, correções e novos recursos para a plataforma.

Nossa missão é construir uma plataforma feita **pela comunidade e para a comunidade**.

---

## 🚀 Tecnologias Utilizadas

<img src="https://skillicons.dev/icons?i=react,nodejs,javascript,html,css,git,github,vscode" />

* ⚛️ **React** — Interface moderna e responsiva.
* 🟢 **Node.js** — Backend rápido e escalável.
* 🤖 **IA Chinesa (MiniMax 3.0)** — Utilizada para auxiliar no desenvolvimento, automações e implementação de novos recursos.
* 🌐 **Open Source** — Evolução contínua através da colaboração da comunidade.
